My modification of [this boilerplate](https://github.com/rokoroku/react-redux-typescript-boilerplate)

## Installation

```
$ npm ci

```


## Running

```
$ npm start
```

## Build

```
$ npm run build
```

## Format code (using all yummies from tslint and stylint)

```
$ npm run lint
```
